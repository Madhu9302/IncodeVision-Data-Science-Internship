# Task 1: Data Cleaning & Preprocessing

## Objective
The objective of this project is to clean and preprocess the Titanic dataset to make it suitable for data analysis and machine learning.

## Dataset
- Titanic Dataset

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy

## Tasks Performed
- Loaded the dataset
- Checked missing values
- Filled missing values
- Removed duplicate records
- Encoded categorical features
- Saved the cleaned dataset

## Files Included
- Task1_Data_Cleaning.ipynb
- Titanic-Dataset.csv
- Titanic_Cleaned.csv

## Output
The cleaned dataset is ready for Exploratory Data Analysis (EDA) and Machine Learning.
